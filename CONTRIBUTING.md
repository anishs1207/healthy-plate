# Contributing to this Project

First off, thank you for considering contributing! It's people like you that make open source such a great community.

## How to Contribute

### 1. Fork & create a branch

If this is something you think you can fix or improve, then fork the repository and create a branch with a descriptive name.

A good branch name would be (where issue #123 is the ticket you're working on): `issue-123-add-new-feature`.

### 2. Local Setup

Make sure you have Node.js installed, then run the following in the project root:

```bash
npm install
```

To run the development server:
```bash
npm run dev
```

### 3. Implement your fix or feature

Make your changes in your branch. We use ESLint and TypeScript for this project, so be sure your code passes the linting checks and builds successfully:

```bash
npm run lint
npm run build
```

### 4. Make a Pull Request

Once you are done with your changes, push your branch to your forked repository and submit a Pull Request! Please include a clear description of the changes and any relevant issue numbers.

## Code of Conduct
Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
